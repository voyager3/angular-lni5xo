To use specific Kendo UI For Angular component you should:
  1. Add the package in npm: ng add @progress/kendo-angular-inputs
  2. Move the added dependencies (InputsModule) from app.module to shared.module in the array kendoUIModules.
  3. Add the component scss in kendo-theme.scss: @import "~@progress/kendo-theme-bootstrap/scss/input";.
  4. Override the component styles in kendo-theme.scss if needed.
